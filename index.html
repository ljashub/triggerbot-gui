FAQ

Is PythonBot safe?
Not 100% safe! Use at your own risk. Nothing is injected into the game and no game files are changed.

Is this a cheat?
It's a comfort tool (triggerbot), not a classic cheat. No unfair advantages like aimbot, wallhack, etc. No injection, no manipulation of the game process.

How can I give feedback?
Via GitHub or our Discord server. We welcome community feedback!
